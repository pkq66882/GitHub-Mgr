GitHub仓库管理工具功能清单：  
**1、用户认证**  
GitHub Token登录  
Token本地存储（自动登录）  
用户菜单（显示用户名/退出）  

**2、仓库管理**  
查看仓库列表  
创建新仓库（名称/描述/私有设置）  
复刻仓库（通过URL）  
删除仓库  

**3、文件操作**  
浏览仓库目录结构（面包屑导航）  
创建文件夹 上传文件（多文件支持）  
下载文件（单个/批量下载为ZIP）
源码下载
删除文件/文件夹（单个/批量）  
重命名文件 
文件类型图标识别  

**4、高级功能**  
静态网站托管（GitHub Pages配置）  
运行工作流（触发GitHub Actions工作流）  
工作流进度监控  
CDN链接生成（jsDelivr代理）  

**5、其它功能**  
返回上级目录  
刷新当前目录  
仓库根目录快捷访问
路径递归快捷访问 

该工具通过GitHub API实现所有功能
